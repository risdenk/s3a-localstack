# s3a-localstack
Testing of Apache Hadoop S3A with Localstack

## Overview
S3A from Apache Hadoop is a way to interact with S3 compatible endpoints. Localstack provides AWS like services for testing.

## Requirements
* Java
* Maven
* Docker

## Testing
* `mvn test`
